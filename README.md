# なかまっする
## 概要
仲間と競い合いながら、筋トレのモチベーションを維持できるアプリです。  
技育CAMPキャラバンin福岡(6/9(金)~6/17(土))にて三人のチーム開発を行い、優秀賞を受賞しました。  
これはそのバックエンドリポジトリです。
